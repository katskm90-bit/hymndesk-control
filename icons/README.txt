Place two PNG icons in this folder before the PWA install prompt will work:

  icon-192.png   (192 x 192 pixels)
  icon-512.png   (512 x 512 pixels)

You can re-use the icons from your HymnDesk project, or generate fresh ones at
https://realfavicongenerator.net using a HymnDesk Control logo.

The app will still run without these icons. They are only needed for the
"Install app" prompt on mobile and desktop.
